### Gerik Features 
* Hand and body pose gesture recognition using mediapipe
* Control mouse and keyboard using hand and body gesture
* Give access for user to get a list of hand and body gesture
<br>coordinates result via websocket
* The local Websocket address is ws://127.0.0.1:2014
* All the process runs locally without internet needed
* You can use Gerik's data if your App / Games need custom
<br>control using gesture recognition
